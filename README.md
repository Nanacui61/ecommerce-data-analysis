# Brazilian E-commerce Data Analysis

Analysis of 99,441 orders from Olist e-commerce platform.

## Key Findings
- Peak orders: 7,544 in November 2017 (Black Friday effect)
- Average delivery time: 12.1 days (median: 10 days)
- Right-skewed delivery distribution indicates outliers pulling mean up

## Skills Used
- Python (Pandas, Matplotlib)
- Data cleaning & transformation
- Time series analysis
- Statistical analysis
